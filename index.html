TrustLens

A Zero Trust Monitoring System

Protect your PC with advanced behavioral analytics and real-time threat detection.

TrustLens is a system monitoring and anomaly detection platform designed to enhance security and reliability in personal computers and organizational environments. The system continuously observes system behavior by collecting real-time metrics such as CPU usage, memory consumption, network activity, disk operations, and process-level resource utilization. By applying machine learning techniques, specifically the Isolation Forest algorithm, TrustLens learns the normal behavioral patterns of a system and detects deviations that may indicate suspicious activities, malware behavior, or system misuse. This monitoring approach helps identify potential threats early and improves overall system awareness.

The system provides real-time visualization of system metrics through live graphs and enables administrators to monitor anomalies across connected systems via a socket-based communication mechanism. Additionally, TrustLens performs automated daily maintenance by generating PDF and Excel reports of detected anomalies, ensuring efficient log management and analysis.
